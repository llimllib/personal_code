K5 Utility Script #1 (Diaries)

By llimllib
Tue May 27th, 2003 at 07:59:18 PM EST

Inside is a script to count the number of diaries, comments, comment ratings, and/or stories by any user of this site. It's only been tested on Linux with Python 2.2 and on mine and rusty's accounts, so there could still be errors. Feel free to comment/criticize, it's appreciated.

Just for the record, rusty has posted 3188 comments, in my program's estimation.

It is available here. I would have put the source of the program here, as it's only 150 lines or so, but I wasn't sure if this is cool. So, if anyone who runs this place would like me to, just say the word and I'll pull it. I just think it's easier than paging through people's (and my own) comment pages to see how many I've posted. Don't recognize a story author's name? just type "python k5_count.py -u AnonymousUser -cds" into a terminal window to see how many comments, diaries and stories they've posted.
